# 스터디 산출물
- 해당 레포지토리에 공부한 내용 업로드
- 최종 산출물로 AI 활용과 관련된 본인만의 경험, 도전 등이 담긴 블로그 포스팅 작성 필수

# 진행방식
- 매주 정해진 분량의 강의를 학습 후 정리하고 서로 리뷰 및 토론하는 방식으로 진행
- 최종 산출물 작성을 위해 개별적으로 디깅중인 내용도 함께 공유

# 과제 제출 방법
- 해당 레포지토리를 clone
- clone 한 프로젝트에 자신의 브랜치를 생성
- 해당 브랜치에서 작업 후 작업물을 생성한 브랜치로 푸시하고 main으로 pr
- 주차가 종료되면 메인으로 각자 푸시

- 매주 정해진 날 스터디 전까지 과제를 제출해야 함.
- 1:1 혹은 2:1 인원에 따라 나눠 모의 면접 형식 스터디 진행

    - 미리 알림 없이 불참 or 과제 미제출 벌금 2000원 차감
